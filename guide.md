## 웹 개발 과정 (프론트엔드 + 백앤드)

# 프로젝트를 위해 필요한 선택지


```
1. 서버선택
2. 백앤드의 언어선택
3. 데이터베이스 선택
4. 통신 방법(아키텍트만들기)
5. 프론트엔드 (SPA, 링크방식)
```

# 기술사항

```
1. OS를 선택한다. ( window server, linux(추천) )
	- window
	- centos (취업추천)
	- ubuntu (추천) - hanpixel
2. 웹서버 ( 웹언어를 해석할수 있게 도와주는 역할이기도 하고 서버의 역할 )
	- apache (취업추천)
	- nginx (떠오르는 샛별) - hanpixel
	- node (자바스크립트) - soket io - hanpixel
3. 백엔드
	- php - hanpixel
	- python (추천) - hanpixel
	- java & jsp (취업추천)
	- node - hanpixel
	- go lang
4. 데이터베이스
	- oracle DB
	- mysql & mariadb (취업추천)
	- mongodb (NoSQL)
5. 개발방식
	- restful api (취업추천) - hampixel
	- MVC ( model - controller - view )
6. 프론트엔드
	- HTML, CSS, JS
7. 프론트엔드 개발방식
	- SPA (react, angular, vue)
	- 페이지이동 방식
8. ui/ux
	- 반응형
	- 하이브리드 (웹+모바일웹)
	- 네이티브 (웹+앱)
```
